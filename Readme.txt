This database contains 477,000+ English language words with their respective parts of speech entries.Those words which have more than 1 parts of speech usage are marked in columns PoS1, PoS2, PoS3 and PoS4.PoS1 column will always return some value (string), afterwards if a column entry is null that means it has no more parts of speech category.




There are two rar files.
One is database file (includes .mdf and .ldf files).
One is Backup file (.bak file).
You can use either one of them.


These database are created using MSSQLS2008.
